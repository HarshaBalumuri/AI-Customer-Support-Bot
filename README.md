# 🤖 AI Customer Support Bot

An AI-powered customer support chatbot built using the MERN Stack.

## ✨ Features

- User Authentication (Signup/Login)
- AI Chat Support
- Chat History
- Product Queries
- Order Status
- Refund Assistance
- Contact Support
- Responsive UI

## 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 🚀 Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🔌 API Endpoints

### Authentication

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/auth/signup | Register a new user |
| POST | /api/auth/login | Login user |

### Chat

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /api/chat | Send a message to the AI chatbot |
| GET | /api/history | Get chat history |
